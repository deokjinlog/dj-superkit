## §5 통합 / 외부 의존

### 5.1 generating-html 호출 체인 (B 항목 — race delay 5초)

변경 전 (v2.3.6):

```
auto-brainstorming Step 4 (.md write)
  → Step 4.5: Agent run_in_background=true (generating-html 호출)
  → Step 5: change-history (footer 1건 추가)
  → Step 6: transition + auto-design invoke
```

`Step 4.5 → Step 5` 사이 race — background subagent 가 .md 읽을 때 footer 1건 인지 가능 → `generating-html` 의 "live doc 진입" 룰 적용 → silent skip.

변경 후 (v2.4):

```
auto-brainstorming Step 4 (.md write)
  → Step 4.5: Agent run_in_background=true (generating-html 호출)
  → Step 4.5+: 5초 delay (race window 확보)
  → Step 5: change-history (footer 1건 추가)
  → Step 6: transition + auto-design invoke
```

5초 delay 안에 background subagent 가 .md 읽기 시작 → footer 0건 시점에 cached → live doc 룰 적용 안 됨 → .html 생성 정상.

### 5.2 silent log (B-3)

`generating-html` SKILL.md 호출 직후 `.js-super/html-regen.log` 에 entry append:

```
2026-05-23 14:00:23 | DISPATCH | <slug>-requirements.md | agent_id=aac604d2fde03f8e1
2026-05-23 14:01:48 | SUCCESS  | <slug>-requirements.html | 41634 bytes
```

`/sync-html --check` 옵션이 마지막 N entry (default 10) 출력. 사용자가 dispatch 결과 확인 가능.

### 5.3 외부 의존 — 변경 0

CDN / 라이브러리 / 외부 도구 추가 없음. 모든 변경은 prompt 본문 + 메인 응답 양식.
