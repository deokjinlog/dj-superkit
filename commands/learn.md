# /learn

이 슬래시는 `learning-notes` skill 을 명시 호출합니다.

> 슬래시 없이 **"이거 학습 정리해줘"** 라고 말해도 자동 발동합니다 — **명령·스킬 둘 다** 됩니다.

## 사용 예

```
/learn                        # 지금까지 대화 내용을 학습노트로 정리
/learn 이 파일 학습 정리해줘     # 지정 자료(코드/문서/URL) 기반
/learn 이 노트에 이어줘         # 같은 챕터면 기존 노트에 누적
```

## 동작

- **입력**: 대화 내용 또는 지정 자료(코드/문서/URL) — 둘 다 지원
- **구조**: 하이브리드 (개요 → 핵심 개념 → 헷갈렸던 점 Q&A → 도식/비유 → 한 줄 요약)
- **저장**: `docs/learning/<날짜>-<주요내용-슬러그>/<슬러그>.md` + 보기 좋은 `.html` 사본
- **누적**: 같은 챕터 후속 학습이면 새 파일 대신 **기존 노트에 이어붙임** (조각남 방지)
- **삭제/정리**: "그 Q&A 빼줘" (부분) / "이 노트 지워줘" (전체) / "이 노트 정리해줘" (중복 정리)

## 범위 밖

복습 알림·검색·태그 X / 과거 대화 소급 정리 X / 개인 로컬 지식 관리용.
